Example description

LED chaser example program for Atmel AVR

This example uses the Atmel AVR STK500 evaluation board together with AVR JTAGICE mkII. The application will alternately flash the User LEDs on the STK500 board.

Follow the steps below to run the example on your hardware.

  1. Connect the computer and the AVR JTAGICE mkII using a USB cable or a serial port cable.
  2. Connect the AVR JTAGICE mkII and the AVR STK500 evaluation board according to the hardware documentation.
  3. Connect the LEDs header with the PORTB header using the 10-wire cable supplied with the STK500 Starter Kit. The cable must not be twisted.
  4. Turn on the power on the AVR JTAGICE mkII and the AVR STK500 board.
  5. Choose Project -> Debug. This will build and download your application to the AVR STK500 board, and then start the debugging session.
  6. To execute the application in the debugger, choose Debug -> Go.
